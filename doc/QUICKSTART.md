# 快速启动指南

## 第一步：安装依赖

```bash
npm install
```

## 第二步：配置API密钥

1. 在项目根目录创建 `.env.local` 文件
2. 添加DeepSeek API密钥（必需）：

```env
DEEPSEEK_API_KEY=your_deepseek_api_key_here
```

3. （可选）添加OpenAI API密钥用于真实图像生成：

```env
OPENAI_API_KEY=your_openai_api_key_here
```

> 详细配置说明请查看 [ENV_SETUP.md](./ENV_SETUP.md)

## 第三步：启动开发服务器

```bash
npm run dev
```

## 第四步：访问应用

打开浏览器访问：http://localhost:3000

## 使用流程

### 生成故事脚本

1. 在首页点击"生成故事脚本"
2. 输入故事描述（可以是主题或详细情节）
3. 点击"生成脚本"，等待AI生成
4. 在对话界面中与AI交互，完善脚本
5. 确认并保存脚本

### 生成漫画绘本

1. 在首页点击"生成漫画绘本"
2. 选择已保存的脚本，或点击"导入外部脚本"
3. 如果脚本超过10页，系统会自动切分为多个片段
4. 选择要生成的脚本片段序号
5. 点击"生成绘本"，等待生成完成

## 注意事项

- 首次使用需要配置DeepSeek API密钥
- 图像生成功能默认使用占位符，如需真实图像请配置图像生成API
- 脚本会自动保存到浏览器本地存储
- 建议在网络良好的环境下使用

## 故障排除

### API调用失败
- 检查API密钥是否正确配置
- 确认网络连接正常
- 查看浏览器控制台的错误信息

### 脚本生成缓慢
- DeepSeek API响应时间取决于网络和服务器负载
- 请耐心等待，通常需要几秒到几十秒

### 图像显示为占位符
- 这是正常现象，除非配置了真实的图像生成API
- 查看 `lib/imageGenerator.ts` 了解如何集成图像生成API
